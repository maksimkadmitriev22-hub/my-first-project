# QuestWorld — сайт квестов в реальности

Многостраничный сайт, созданный с нуля без использования CMS.

## Стек
- HTML5
- CSS3 (Flexbox, Grid, анимации, 3D-трансформации)
- JavaScript (анимация счётчиков, карусель, аккордеон, модальное окно)

## Основные фичи
- 3D-карусель отзывов
- Анимированные счётчики с запуском при прокрутке
- Модальное окно бронирования
- FAQ-аккордеон
- Адаптивная вёрстка
- Навигация с якорями
